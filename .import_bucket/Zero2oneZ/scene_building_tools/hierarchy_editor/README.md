# Hierarchy Editor

The Hierarchy Editor provides a tree view of the scene graph with parent-child relationships.

## Features

- Tree view of scene objects
- Drag and drop reparenting
- Multi-selection operations
- Visibility and lock toggles
- Search and filtering
- Context menu for common operations
- Instance indicators
- Custom sorting and organization
- Collapsible groups and folders
- Color coding and icons

## Implementation

The Hierarchy Editor is implemented as a dockable panel with a tree structure. It provides a structural view of the scene and allows for quick selection and organization of objects.

## Technical Requirements

- Efficient handling of large scene hierarchies
- Custom drawing for specialized node types
- Undo/redo for structural changes
- Support for prefabs and instances
