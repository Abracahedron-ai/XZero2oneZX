# Scene Building Tools

This directory contains tools for constructing and manipulating 3D scenes within the Zero2oneZ platform.

## Components

- **Asset Shelf**: Library of assets that can be added to the scene
- **Hierarchy Editor**: Tree view of the scene graph with parent-child relationships
- **Property Editor**: Interface for editing object properties and attributes
- **Viewport**: 3D view of the scene with navigation and manipulation tools

## Features

- Complete 3D transformation stack
- Parent/child relationships with constraints
- Selection and grouping tools
- Pivot manipulation
- Transform spaces (World, Local, Parent, View)
- Snapping and alignment tools
- Layer and visibility management

## Integration Points

The scene building tools integrate with:
- Animation Tools for animating scene objects
- AI Agents for scene analysis and optimization
- VFX Tools for adding effects to the scene
